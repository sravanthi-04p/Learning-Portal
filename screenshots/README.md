# Screenshots

Screenshots could not be auto-generated in this sandboxed build environment
(no headless browser available here). To capture them for submission:

1. Start the backend: `cd backend && npm start`
2. Start the frontend: `cd frontend && npm run dev`
3. Open http://localhost:5173 and capture:
   - Register / Login page
   - Home page (video grid + Continue Watching)
   - Video player page with the bookmark panel and 2-3 bookmarks added
   - The screenshot-protection warning overlay (trigger by pressing PrintScreen
     or Cmd/Ctrl+Shift+I while a video is playing)
   - Mobile/responsive view (resize browser or use dev tools device toolbar)

Save images here as .png files before pushing to your GitHub repo.
